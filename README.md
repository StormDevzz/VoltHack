## RU

VoltHack — клиентский мод для Minecraft от организации StormDevzz.
Предоставляет набор модулей для улучшения игрового опыта: combat, movement, render, player, world.

### Возможности

- Aura — автоматическая атака ближайших сущностей
- Velocity — уменьшение отдачи
- NoFall — защита от урона при падении
- FullBright — полная яркость
- ESP — подсветка сущностей
- ElytraSwap — быстрая смена элитры и нагрудника
- AutoFarm — автоматическая сборка урожая
- DiscordStatus — отображение статуса игры в Discord Rich Presence
- HUD Editor — редактирование элементов интерфейса

### Сборка

```bash
./gradlew build
```

Готовый JAR находится в `build/libs/`.

### Установка

1. Установите Fabric Loader
2. Поместите JAR в папку `mods`
3. Запустите игру

## EN

VoltHack is an open-source Minecraft client modification by StormDevzz organization.
Provides a set of modules to enhance gameplay: combat, movement, render, player, world.

### Features

- Aura — automatically attacks nearest entities
- Velocity — reduces knockback
- NoFall — prevents fall damage
- FullBright — full brightness
- ESP — highlights entities
- ElytraSwap — quick elytra/chestplate swap
- AutoFarm — automatic crop harvesting
- DiscordStatus — displays game status via Discord Rich Presence
- HUD Editor — rearrange and toggle HUD elements

### Building

```bash
./gradlew build
```

The built JAR will be in `build/libs/`.

### Installation

1. Install Fabric Loader
2. Place the JAR in your `mods` folder
3. Launch the game
